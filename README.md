# Signup Login API

Projeto de estudo usando Spring Boot para implementar um sistema simples de cadastro e login.

## Tecnologias utilizadas
- Java
- Spring Boot
- PostgreSQL
- JPA/Hibernate

## Funcionalidades
- Controller -> Recebe requisições
- Repository -> Comunicação com banco
- Model -> Entidade do banco
- DTO -> Transferência de dados

## Aprendizados
- Tratamento de erros HTTP
- Uso de ResponseStatusException
- Controle de tentativas de login
- Integração com PostgreSQL
